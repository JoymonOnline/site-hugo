Changelog

Here I write the history and evolution of my web site. It helps me look at my past and analyze how far I improved my site.

v7

08Nov2020
Migrated from AngularJS 1.x to JAMstack using Hugo. Staging version ready.

v6

26Jan2020
Enabled http(s) and marked the site http(s) only from GitHub pages.
24Jan2020
Started using TravisCI to deploy to production environment.
17Oct2018
Separated PrayerBook app to its own hosting space and redirected from here.
13Mar2017
Serves the site from Github Pages keeping only domain with GoDaddy.
08Feb2017
Converted to Serverless Architecture. Started using AngularJS 1.x as JavaScript framework at browser.

v5

09May2016
Started using FontAwesome library for icons.
05May2016
Made the site responsive without bootstrap.
29Oct2015
Moved source to Github and setup continuous integration and delivery. Used AppVeyor for CI and CD
